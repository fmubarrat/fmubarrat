### Featured Research

#### [Parkinson's Motor Assessment — Drawing Product to Process](https://github.com/fmubarrat/Parkinsons-Hand-Kinematic-Analysis-Smartphone)

<table>
<tr><td>

**Video-Based 3D Hand Kinematics for PD Detection**

A two-part framework combining 2D sketch analysis (204 clinical drawings) with novel 3D hand tracking during drawing using only a smartphone camera.

| Metric | Result |
|--------|--------|
| Micrographia Detection | **p=0.019**, Cohen's d=3.2 |
| Tremor Feature Increase | **+2101%** (normalized jerk) |
| PD-Band Power Increase | **+1431%** |
| Classification (Kaggle) | **69.7%** SVM accuracy |
| Hardware Cost | **$0** (phone camera only) |

**Stack:** `MediaPipe` `SciPy` `scikit-learn` `OpenCV` `FFT/SPARC`

*Manuscript in preparation, 2025*

</td></tr>
</table>
